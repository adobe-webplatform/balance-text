Balance Text Demo Page

The left and right columns in the table below have identical content and markup, except that the solution is applied to the right side, so compare the rendering between the two sides.

To test how the algorithm works in different text wrapping cases, open this page in any modern browser, and then do either of the following to see how text rendering differs between the two sides:

  • Change the width of the browser window.
  • Change the font size using Ctrl "+" and Ctrl "-".

Note that due to how different browsers send out resize events, you may need to refresh the browser to see the balanced text solution on the right.
